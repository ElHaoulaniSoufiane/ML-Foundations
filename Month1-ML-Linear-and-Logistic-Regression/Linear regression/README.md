# linear regression from scratch

i'm learning machine learning and decided to build everything from scratch instead of just using sklearn.  
no shortcuts — just python, numpy, and math.

this is week 1. one notebook per day.

---

## why from scratch?

using a library is easy. but if you don't know what's happening underneath, you're just calling functions.  
i want to actually understand what the model is doing — so i'm building it step by step.
